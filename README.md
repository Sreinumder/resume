Resume
----------------
Modified version of [NNJR](https://github.com/tzx/NNJR) to my personal needs and likings.
Requires [Typst](https://github.com/typst/typst) for compilation.

Usage
----------------
1. Edit resume.typ fields to your liking
2. Compile
```bash
typst compile resume.typ
```
3. Open Resume.pdf!
